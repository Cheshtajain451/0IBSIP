Temperature Converter

This project was developed as part of the Oasis Infobyte Web Development Internship (Level 1). The objective of this project was to create a simple, responsive and user-friendly temperature converter using HTML5, CSS3 and JavaScript. The application allows users to convert temperature values between Celsius, Fahrenheit and Kelvin quickly and accurately.

While developing this project, I focused on creating a clean interface, writing organized code and providing a smooth user experience. The design is responsive and works properly on desktops, tablets and mobile devices. JavaScript was used to perform the temperature conversion instantly based on the user's input.

This project helped me improve my understanding of JavaScript fundamentals, DOM manipulation, user input handling, mathematical calculations and responsive web design. It also gave me practical experience in building interactive web applications using frontend technologies.

The converter validates user input and displays accurate conversion results in real time. The layout is simple, easy to understand and designed to provide a better user experience.
